#### Overview

`pool-day` is a simple thread pool library for C/C++ applications.

#### Build

```cmake
$ cmake -S . -B build
$ cmake --build build
```

Optional build flags:

|        Flag         |                 Description            |
|---------------------|----------------------------------------|
|    BUILD_SAMPLES    | Build the library samples              |
|   ENABLE_LOGGING    | Enable the library logging feature     |

#### Support

- linux
