## pool-day

#### Overview

`pool-day` is a thread pool library for C/C++ applications.

#### Build

```cmake
$ cmake -S . -B build
$ cmake --build build
```

Optional build flags:

|        Flag         |                 Description            |
|---------------------|----------------------------------------|
|    BUILD_SAMPLES    | Build the library samples              |
|   ENABLE_LOGGING    | Enable library logs                    |

#### Support

- linux
